# 天空之隙 · Sky Gap

宫崎骏式风格个人主页 — 粒子动画 · 视差云层 · 黑胶唱片播放器。

A Miyazaki-inspired personal homepage with particles, parallax clouds, and a vinyl music player.

## 预览 Preview

打开 `index.html` 即可在浏览器中查看。

## 技术栈 Tech

- 纯 HTML / CSS / JavaScript
- Canvas 光粒子系统
- CSS 视差云层 + 漂移动画
- 鼠标跟随柔光
- 黑胶唱片音乐播放器

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/Guyugjn/sky-gap-homepage.git

# 将音乐文件放入 assets/music/ 目录（命名 00.mp3 ~ 62.mp3）
# 然后直接用浏览器打开 index.html
```

> 音乐文件未包含在仓库中，需自行放入 `assets/music/`。
